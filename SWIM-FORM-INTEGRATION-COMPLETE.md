### 2. **movelapGenerator.ts** ✅
**Location**: `src/utils/movelapGenerator.ts`

**Functions**:
- `generateMovelaps()` - Creates movelaps from sequences
- `generateMoveframeDescription()` - Creates description string
- `calculateTotalReps()` - Sums reps across sequences
- `validateSequence()` - Validates individual sequence
- `validateSequences()` - Validates all sequences

**Example Usage**:
```typescript
const movelaps = generateMovelaps({
  moveframeId: 'xxx',
  sport: 'SWIM',
  sequences: [
    { meters: 400, repetitions: 6, speed: 'A2', style: 'Freestyle', pause: "1'30\"", endPause: "5'" },
    { meters: 200, repetitions: 3, speed: 'B1', style: 'Freestyle', pause: "1'" }
  ],
  globalFields: {
    pace100: '1:30/100',
    alarm: -3,
    sound: 'Beep'
  }
});
// Returns 9 movelaps (6 + 3)

const description = generateMoveframeDescription('SWIM', sequences);
// Returns: "400m x 6 A2 Freestyle pause 1'30\" + 5' + 200m x 3 B1 Freestyle pause 1'"
```

### 3. **create-with-movelaps API** ✅
**Location**: `src/app/api/workouts/moveframes/create-with-movelaps/route.ts`

**Endpoint**: `POST /api/workouts/moveframes/create-with-movelaps`

**Features**:
- ✅ Creates moveframe + all movelaps in one transaction
- ✅ Validates all required fields
- ✅ Checks workout session exists
- ✅ Checks section exists
- ✅ Atomic operation (all or nothing)

**Request Body**:
```json
{
  "workoutSessionId": "xxx",
  "sectionId": "xxx",
  "letter": "A",
  "sport": "SWIM",
  "type": "STANDARD",
  "description": "400m x 6 A2 Freestyle pause 1'30\" + 5' + 200m x 3 B1 Freestyle pause 1'",
  "movelaps": [
    { "repetitionNumber": 1, "distance": 400, "speed": "A2", "style": "Freestyle", "pause": "1'30\"", "status": "PENDING" },
    { "repetitionNumber": 2, "distance": 400, "speed": "A2", "style": "Freestyle", "pause": "1'30\"", "status": "PENDING" },
    // ... 7 more movelaps
  ]
}
```

**Response**:
```json
{
  "success": true,
  "moveframe": { /* moveframe object */ },
  "movelaps": [ /* array of 9 movelaps */ ]
}
```
